# Requirements
* Users <b>must</b> be able to create/delete/edit an account
* Users <b>must</b> be able to sign in or log out, and they should be able to request a password reset if needed.
* Users <b>must</b> be able to post/edit/remove their posts.
* Users <b>must</b> be able to view posts and filter/sort through these.
* Users <b>must</b> be able to vote on posts (1 vote per post + can change vote)
* Users <b>must</b> be able to comment on posts if signed in / registered
* Users who choose not to sign in <b>must</b> be able to post anonymously and vote.
* Admins <b>must</b> be able to remove anyone's post if inappropriate
* App <b>must</b> display results of poll voting to all users. 
* App <b>must</b> display posts with most total votes
* Each post <b>must</b> be tagged with a category (Social, Work, Health, Money etc)
* The app could provide the user a notification feed to show updates on their post’s progress when logged in.
